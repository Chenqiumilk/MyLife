# App Store Checklist

## Product

- [ ] Define MVP scope.
- [ ] Remove placeholder or non-functional settings.
- [ ] Confirm pricing model.
- [ ] Add real StoreKit purchase flow before showing final price claims.

## Engineering

- [ ] Build with current required Xcode and iOS SDK.
- [ ] Use persistent local storage.
- [ ] Store photos outside database when appropriate.
- [ ] Handle large image uploads.
- [ ] Test app restart and upgrade persistence.
- [ ] Test small and large iPhones.
- [ ] Test dark mode and dynamic type.

## Privacy

- [ ] Write privacy policy.
- [ ] Document photo library usage.
- [ ] Document notification usage.
- [ ] Confirm whether any analytics or third-party SDKs collect data.
- [ ] Fill App Store privacy details.

## App Store Connect

- [ ] App name.
- [ ] Bundle ID.
- [ ] SKU.
- [ ] Category.
- [ ] Age rating.
- [ ] Screenshots.
- [ ] App description.
- [ ] Keywords.
- [ ] Support URL.
- [ ] Privacy policy URL.

## Testing

- [ ] TestFlight internal testing.
- [ ] External beta testing.
- [ ] Crash review.
- [ ] App Review notes.
