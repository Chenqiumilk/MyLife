# Design Notes

## Aesthetic

The app should feel:

- Korean romantic cute.
- Soft and low-saturation.
- Hand-drawn.
- Calm rather than productivity-heavy.
- Personal and private.

## Home Calendar

- Minimal monthly calendar.
- Past days can show mood icons.
- Future days show numbers.
- Mood icons should be varied: cloud, heart, egg, bear, dog, apple, flower, star.

## Diary Page

- Diary writing should feel like one continuous page.
- Timestamp moments should be inline with diary writing.
- Photos should appear in one frame and support horizontal swiping.
- Todos should be visually separate but lightweight.

## Focus Page

- Background should use cozy room/cafe/library imagery.
- Pet should feel like a companion inside the scene.
- Pet animation should include walking, blinking, and breathing.
- Timer controls should stay simple.

## Avoid

- Heavy dashboard styling.
- Corporate productivity UI.
- Overly saturated colors.
- Too many cards inside cards.
