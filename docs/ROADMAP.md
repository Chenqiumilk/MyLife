# Roadmap

## Phase 0: Repository and Prototype

- Organize GitHub-ready project structure.
- Keep web preview runnable on Windows.
- Document product and design direction.
- Capture screenshots and demo GIFs.

## Phase 1: Native iOS MVP

- Rebuild core flows in SwiftUI.
- Add SwiftData/Core Data models.
- Store photos in app file storage.
- Add focus session persistence.
- Add settings state.
- Test on iPhone simulator and physical device.

## Phase 2: TestFlight

- Add app icon and launch screen.
- Add onboarding.
- Add privacy policy.
- Add crash-free testing with internal testers.
- Validate diary/photo/focus workflows.

## Phase 3: App Store Readiness

- Create App Store Connect record.
- Prepare screenshots, description, subtitle, keywords, age rating.
- Fill privacy nutrition label.
- Add StoreKit if paid version is included.
- Submit for review.

## Phase 4: Portfolio and Launch

- Write product case study.
- Create portfolio project page.
- Add demo video.
- Share roadmap and release notes.
