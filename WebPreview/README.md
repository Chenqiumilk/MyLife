# Diary Focus Web Preview

Open `D:\PostPhD\diaryapp\WebPreview\index.html` in Chrome or Edge on Windows.

You can also run it from PowerShell:

```powershell
Start-Process .\WebPreview\index.html
```

This is a local browser prototype of the iOS app concept. It previews the calendar, mood picker, diary editor, picture carousel, inline timestamped diary moments, daily todos, settings, review pages, animated focus companion picker, and focus timer.

Diary text, selected moods, uploaded diary pictures, timestamp notes, todos, and the selected focus pet are saved in browser local storage for this preview. Clearing site data for the file or using another browser will clear the preview data.

Book search uses Open Library. Movie search uses Apple's public iTunes Search API. Both are no-key APIs for this preview.
